# LocalConnect - Multi-Vendor Marketplace for Local Products in Nepal

**Frontend of LocalConnect**, a multi-vendor e-commerce platform connecting local suppliers, vendors, and customers in Nepal. The platform promotes authentic local products such as handicrafts, textiles, agricultural goods, and souvenirs.

## Project Overview

LocalConnect solves the problem of limited market access for local producers in Nepal by providing a web-based multi-vendor platform. It supports:

- Category-based product listing (handicrafts, textiles, food items, etc.)
- Vendor-mediated order flow
- Supplier variant system
- Role-based dashboards for Customers, Vendors, and Admins

---

## Features

- Browse and search products
- Add products to cart
- User registration and login
- Vendor order management
- Admin dashboard for product and user management

---

## Tech Stack

- **Frontend:** ReactJS, Tailwind CSS
- **Backend:** Spring Boot (Java) - separate repo
- **Database:** MySQL
- **Version Control:** Git & GitHub
- **Other Tools:** VS Code, Postman
